# Heart-Attack-Prediction-App
A Machine Learning based web application that predicts the risk of heart attack using patient health data. Built using Python, Scikit-learn and Streamlit.
# ❤️ Heart Attack Prediction App

## 📌 Project Overview
This project is a Machine Learning-based web application that predicts the risk of heart attack using patient health data.

---

## 🚀 Features
- User-friendly interface  
- Real-time prediction  
- Displays probability of heart attack risk  

---

## 🧠 Technologies Used
- Python  
- NumPy  
- Scikit-learn  
- Streamlit  
- Joblib  

---

## 📊 Dataset
The dataset was taken from Kaggle and includes features like:
- Age  
- Sex  
- Cholesterol  
- Blood Pressure  
- Heart Rate  

---

## ⚙️ Steps Performed
1. Data Collection from Kaggle  
2. Data Cleaning & Preprocessing  
3. Outlier Detection & Removal  
4. Feature Encoding & Scaling  
5. Model Training using Random Forest  
6. Model Evaluation & Accuracy Improvement  
7. Model Saving using Joblib  
8. Web App Development (Flask → Streamlit)
